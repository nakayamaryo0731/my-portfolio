# Suggested Commands

## Development
```bash
pnpm dev        # 開発サーバー起動 (astro dev)
pnpm start      # 開発サーバー起動 (alias)
pnpm preview    # プロダクションビルドのプレビュー
```

## Build
```bash
pnpm build      # プロダクションビルド
```

## Code Quality
```bash
pnpm eslint     # ESLint実行 (src/**/*.{js,astro})
pnpm check      # Astro型チェック (astro check)
```

## System Commands (macOS/Darwin)
```bash
git status      # Gitステータス確認
git diff        # 変更差分確認
git log --oneline -10  # 最近のコミット履歴
```

## Cloudflare (Wrangler)
```bash
pnpm wrangler pages dev dist  # ローカルでCloudflare環境テスト
```
